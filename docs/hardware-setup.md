# HermesLens — Hardware Setup

## What you need

- M5 StickS3
- USB-C data cable

## Physical overview

The M5 StickS3 is an all-in-one stick form factor. It has:

- 240×135 display
- Side button
- Front button
- USB-C port (power and data)

## Setup

1. Plug the M5 StickS3 into the USB-C cable.
2. Connect the other end to a power source or computer.
3. Use the side button and front button to navigate the dashboard.
